# HTML Table
## Table is a combination of row and cols

> ### Table represent by  &lt; table &gt; &lt; /table &gt;

> ### Row represent by  &lt; tr &gt; &lt; /tr &gt;

> ### Columns represent by  &lt; td &gt; &lt; /td &gt;

## Table Attribute

- border
- width
- height
- cellspacing
- cellpadding
- bg-color
- align

## TD Attribute

- width
- height
- colspan
- rowspan
- bg-color
- align
- valign
